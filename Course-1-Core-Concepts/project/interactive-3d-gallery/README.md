# Project: Interactive 3D Gallery

A complete interactive gallery showcasing Course 1 concepts!

## Features

- Multiple clickable 3D objects
- Hover effects with animations
- Selection system
- Smooth interactions
- Professional presentation

## Concepts Demonstrated

- Event handling (click, hover)
- useFrame animations
- State management
- Refs for control
- Material properties
- Scene composition

## Running

```bash
npm install
npm run dev
```
